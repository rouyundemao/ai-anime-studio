# AI幻梦工作室网页优化方案

**创建时间**: 2026-04-23  
**优化目标**: 为每个创作模块嵌入 AI 落地教学，实现「艺术理念→AI 操作→成果落地」闭环

## 三、组件设计规范

### 1. PromptTemplate 组件

**功能**: 显示可复制的 Prompt 模板，支持一键复制

**Props**:
```typescript
interface PromptTemplateProps {
  title: string           // 模板标题
  description?: string    // 模板说明
  prompt: string          // Prompt 内容
  tool?: string           // 适用工具（如 Midjourney, SD, Claude）
  category?: string       // 分类（如 留白艺术、意境营造）
}
```

**UI 设计**:
```
┌─────────────────────────────────────────┐
│ 📋 留白艺术 Prompt          [复制按钮] │
├─────────────────────────────────────────┤
│ 适用工具：Midjourney                    │
│                                         │
│ [主体描述], 东方水墨画留白艺术，        │
│ 计白当黑，大面积留白，留足想象空间，    │
│ 极简构图，意境悠远，动漫风格，          │
│ 电影级画质，8K 超高清                   │
└─────────────────────────────────────────┘
```

**交互**:
- 点击复制按钮 → 复制 Prompt 到剪贴板 → 显示"已复制"提示（2 秒后消失）
- 支持代码块格式显示，保持换行和缩进

## 四、各模块 AI 教学内容详细设计

### 模块 1: 艺术理念

**能力锚点**: 东方美学与西方美学的 AI 落地实操

**适配工具**: Midjourney, Stable Diffusion, ChatGPT-4o, 剪映 AI

**分步教学**:

#### ① 东方美学 AI 落地
- **留白艺术**
  - Prompt 模板：`[场景/主体], 东方水墨画留白艺术，计白当黑，大面积留白，留足想象空间，极简构图，意境悠远，动漫风格，电影级画质，8K 超高清`
  - 工具：Midjourney v6
  - 参数：`--ar 16:9 --style raw --s 250`

- **意境营造**
  - Prompt 模板：`[核心情节与情绪], 动漫场景，情景交融，虚实相生，通过环境氛围烘托人物情绪，不着一字尽得风流，东方美学意境，电影级光影，氛围感拉满`
  - 工具：Midjourney / SD

- **韵律节奏**
  - Prompt 模板：`为 [剧本片段] 设计一组动漫分镜，共 4 个镜头，通过景别切换、运镜快慢、画面疏密，实现气韵生动、动静相宜的韵律节奏，标注每个镜头的时长与运镜方式`
  - 工具：ChatGPT-4o

#### ② 西方美学 AI 落地
- **黄金比例与黄金分割**
  - Prompt 模板：`[画面主体], 构图遵循黄金分割与黄金螺旋法则，主体位于黄金分割点，画面平衡有张力，电影级构图，动漫风格，8K 超高清`
  - 工具：Midjourney

- **戏剧张力与三一律**
  - Prompt 模板：`帮我写一个符合三一律原则的动漫短剧剧本，时间、地点、情节高度统一，全程设置悬念，起承转合完整，戏剧张力拉满，时长 3 分钟`
  - 工具：Claude 3 / ChatGPT-4o

- **电影语言与蒙太奇**
  - Prompt 模板：`为 [核心情节] 设计一组蒙太奇动漫分镜，通过镜头组接实现叙事与情绪表达，标注每个镜头的画面内容、时长、声画对位设计，符合电影级蒙太奇语言`
  - 工具：ChatGPT-4o

**成果校验标准**:
- 生成 1 张融合东方留白与西方黄金比例的动漫概念图
- 生成一组 4 镜头的蒙太奇分镜
- 作品符合东方美学意境 + 西方戏剧张力

### 模块 2: 故事创作艺术

**能力锚点**: AI 剧本创作从 0 到 1 全流程

**适配工具**: Claude 3 Opus, ChatGPT-4o, Final Draft AI

**分步教学**:

#### ① 创意与选题生成
- Prompt 模板：`帮我生成 10 个 [题材类型] 的动漫短片创意，每个创意包含核心主题、核心冲突、人物设定，要求主题有深度，能引发观众情感共鸣，有差异化的创意亮点`

#### ② 标准化剧本结构搭建
- 使用「救猫咪 15 节拍表」框架
- Prompt 模板：`帮我按照好莱坞救猫咪 15 节拍表，创作一个 [题材类型] 动漫短片的完整剧本框架，明确标注每个节拍的核心事件、时长占比、情绪峰值`

#### ③ 人物弧光深度设计
- Prompt 模板：`基于以上剧本框架，为每个核心角色设计完整的人物小传，包含内在需求与外在目标，以及人物在每个情节节点的心理转变与成长`

#### ④ 剧本打磨与对白优化
- Prompt 模板：`优化以上剧本的对白与情节节奏，剔除冗余内容，强化情感张力，确保每一句对白都符合人物性格，推动情节发展`

**配套资源**:
- 全类型动漫剧本 Prompt 模板库
- 经典动漫剧本拆解框架
- 剧本完成度自查清单

**成果校验标准**:
- 完成一个 3 分钟动漫短片的完整剧本
- 具备完整的人物弧光、清晰的情感节奏、电影级剧本结构

### 模块 3: 角色设计体系

**能力锚点**: AI 动漫角色全流程设计标准化

**适配工具**: Stable Diffusion (ComfyUI), Midjourney, Kohya_ss

**分步教学**:

#### ① 人物人设与基础设定生成
- Prompt 模板：`创作一个 [年龄/性别/气质] 的动漫角色人设，包含基本信息、性格特点、背景故事、外貌特征、服装风格，确保人物设定立体饱满，有独特的辨识度`

#### ② 五官与形体精准设计
- Prompt 模板：`[人物基础设定], 精致五官，符合黄金分割比例，东方美学骨相，[风格定位], 清晰的面部轮廓，灵动的眼神细节，皮肤质感细腻，无五官变形，无面部崩坏 --ar 3:4 --style raw`

#### ③ 发型与服装体系化创作
- Prompt 模板：`以上人物，保持五官、脸型、气质完全不变，设计 3 套不同场景的 [风格] 发型与服装，每套服装搭配对应的配饰，风格统一，细节完整`

#### ④ 人物一致性终极解决方案（LoRA 训练）
- 工具：Kohya_ss
- 步骤：
  1. 准备 8-15 张同人物的标准图
  2. 参数设置：batch size 2, epoch 10, 学习率 1e-4
  3. 训练完成后即可生成 100% 统一的人物画面

#### ⑤ 标准化角色素材生成
- 三视图 Prompt：`[人物完整设定], 动漫角色三视图，分别为正面视图、侧面视图、背面视图，同一人物，保持五官、发型、服装完全一致，纯白背景，标准平光，无阴影，无透视变形，8K 超高清 --ar 16:9`
- 表情库：8 组基础表情（喜、怒、哀、乐、惊、恐、悲、中性）

**配套资源**:
- 角色设计 Prompt 模板库
- LoRA 训练标准参数预设
- 人物一致性控制全攻略

**成果校验标准**:
- 生成完整的角色人设卡、三视图、8 组基础表情
- 完成专属人物 LoRA 模型训练
- 实现人物的标准化设计

### 模块 4: 世界构建方法

**能力锚点**: AI 世界观全流程构建沉浸式教学

**适配工具**: Claude 3 Opus, World Anvil AI, Inkarnate, Midjourney

**分步教学**:

#### ① 世界观底层框架搭建
- Prompt 模板：`帮我构建一个 [题材类型] 的动漫世界观，包含完整的地理生态设定、文明文化体系、历史神话脉络、社会结构规则，所有设定逻辑自洽，无前后矛盾，贴合东方美学的意境与底蕴`

#### ② 文明文化与历史神话构建
- Prompt 模板：`基于以上世界观，补充完整的世界历史大事件年表、核心文明的神话传说体系、不同势力的纷争与关系，为世界观赋予厚度与沉浸感`

#### ③ 世界观可视化落地
- **地理地图生成**: 使用 Inkarnate 的 AI 地图生成功能
- **场景概念生成** Prompt：`[世界观场景详细设定], 符合以上世界观的动漫场景概念图，电影级画质，三层光影，丰富的环境细节，沉浸感强，8K 超高清 --ar 16:9`

#### ④ 世界观设定标准化归档
- Prompt 模板：`将以上所有世界观设定，整理成标准化的世界观百科文档，按地理设定、文明体系、历史神话、社会结构、核心角色、核心场景分类归档，逻辑清晰`

**配套资源**:
- 世界观搭建 Prompt 模板库
- 经典动漫世界观拆解参考
- AI 地图生成全教程

**成果校验标准**:
- 生成完整的动漫世界观设定集
- 配套世界高清地图、3 个核心场景的概念图
- 完成世界观的全流程构建

### 模块 5: 画面生成

**能力锚点**: AI 电影级动漫画面生成全流程

**适配工具**: Midjourney, Stable Diffusion (ComfyUI), Topaz Gigapixel AI

**分步教学**:

#### ① 电影级镜头语言设计
- 教学内容：景别、运镜、构图的核心逻辑
- Prompt 模板：`[画面主体], 电影级镜头语言，[景别：特写/中景/全景], [运镜：推/拉/摇/移], 构图遵循黄金分割，动漫风格，8K 超高清`

#### ② 三层光影精准控制
- 教学内容：环境光、轮廓光、补光的三层光影设计
- Prompt 模板：`[场景描述], 三层光影：环境氛围光 + 人物轮廓光 + 面部细节补光，电影级色彩分级，8K 超高清，发丝级细节刻画，极致纹理质感`

#### ③ 画质与细节精度提升
- 工具：Topaz Video Enhance AI
- 参数：放大倍率 4-8 倍，降噪强度 30%, 细节增强 40%, 去锯齿开启
- 解决：画面模糊、细节缺失、边缘锯齿问题

#### ④ 电影级调色与风格统一
- 工具：DaVinci Resolve AI 色彩匹配
- 步骤：导入参考帧 → 30 秒完成 1:1 色调匹配 → 开启 AI 肤色保护

**配套资源**:
- 电影级动漫画面 Prompt 库
- 光影与调色关键词库
- Stable Diffusion 电影级画面专属工作流

**成果校验标准**:
- 生成 5 个镜头的动漫分镜画面
- 每个画面符合电影级画质、三层光影、发丝级细节

## 六、技术细节

### Prompt 复制功能实现
```typescript
const copyToClipboard = async (text: string) => {
  try {
    await navigator.clipboard.writeText(text)
    // 显示成功提示
    setCopied(true)
    setTimeout(() => setCopied(false), 2000)
  } catch (err) {
    // 降级方案
    const textarea = document.createElement('textarea')
    textarea.value = text
    document.body.appendChild(textarea)
    textarea.select()
    document.execCommand('copy')
    document.body.removeChild(textarea)
  }
}
```
