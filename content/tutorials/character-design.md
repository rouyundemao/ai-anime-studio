# 角色设计秘籍

> 打造独特动漫角色的秘密武器，包含五官比例、服装设计、表情库等专业技巧。

## 🎨 第一章：角色基础

### 1.1 五官比例黄金法则

动漫角色的五官比例是塑造美感的关键：

| 特征 | 比例 | 描述 |
|------|------|------|
| 眼睛位置 | 头部 1/2 处 | 标准动漫比例 |
| 眼距 | 1.5 个眼睛宽度 | 自然协调 |
| 鼻子高度 | 眉间到下巴 1/3 | 精致小巧 |
| 嘴唇位置 | 下巴到鼻子 1/3 | 柔和自然 |

:::tip 提示词模板
```
完美精致五官，五官比例协调，大眼睛明亮有神，
柳叶眉，樱桃小嘴，皮肤白皙细腻，面部光照均匀
```
:::

### 1.2 发型设计技巧

发型是角色个性的重要表达：

#### 常见发型类型

1. **长发系**
   - 直长发：温柔、知性
   - 卷发：活泼、浪漫
   - 高马尾：干练、元气

2. **短发系**
   - 齐耳短发：俏皮、可爱
   - 波波头：时尚、潮流
   - 蘑菇头：天真、无邪

## 👗 第二章：服装设计

### 2.1 风格分类

#### 古风系
- 汉服：曲裾、襦裙、深衣
- 配饰：发簪、步摇、玉佩
- 元素：刺绣、流苏、飘带

#### 现代系
- 学院风：制服、领带、徽章
- 休闲风：卫衣、牛仔、球鞋
- 职场风：西装、衬衫、高跟鞋

## 😊 第三章：表情库设计

### 3.1 基础表情

| 表情 | 描述 | 使用场景 |
|------|------|----------|
| 开心 | 眼角上扬，嘴角上翘 | 喜悦、满足 |
| 悲伤 | 眉头紧锁，嘴角下垂 | 失落、难过 |
| 惊讶 | 眼睛睁大，嘴巴张开 | 意外、震惊 |
| 愤怒 | 眉毛倒竖，嘴角紧闭 | 生气、不满 |

## 🛠️ 实战练习

### 练习一：设计一个古风角色

:::practice
**任务**：设计一个西汉时期风格的古风美女

**要求**：
- 身材纤细苗条
- 穿曲裾袍
- 乌黑长发高盘成汉代发髻
- 佩戴金钗和步摇

**提示词**：
```
一位西汉时期风格的古风美女，身材纤细苗条，
身穿华丽的红色曲裾袍，衣袖宽大，精致妆容，
柳叶眉，凤眼，朱唇，皮肤白皙，
乌黑长发高盘成汉代发髻，佩戴金钗和步摇，
背景是西汉宫殿，漆器屏风，华丽典雅，
高质量动漫风格，细节丰富
```
:::
