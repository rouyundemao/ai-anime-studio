## 📐 第二章：参数详解

### 2.1 即梦AI 参数

#### 核心参数

```
--prompt: 描述动作和镜头运动
--duration: 视频时长（1-5秒）
--resolution: 分辨率（720P/1080P）
--motion_strength: 动作强度（0.1-1.0）
```

#### 提示词技巧

:::code-group
```bash [镜头推近]
镜头缓慢推近，角色表情逐渐清晰，眼神流露情感

# 参数
--duration 5
--motion_strength 0.3
```

```bash [镜头摇摄]
镜头从左向右缓慢摇摄，展示全景环境

# 参数
--duration 3
--motion_strength 0.5
```

```bash [角色动作]
角色缓慢眨眼，头发随风轻轻飘动，嘴角微微上扬

# 参数
--duration 5
--motion_strength 0.2
```
:::

### 2.2 可灵AI 参数

#### 高级控制

```
--video_duration: 视频时长（5s/10s/30s/60s/120s）
--aspect_ratio: 宽高比（1:1/16:9/9:16）
--video_resolution: 分辨率（720P/1080P）
--camera_move: 镜头类型
```

#### 镜头类型详解

| 类型 | 描述 | 适用场景 |
|------|------|----------|
| zoom_in | 推近 | 强调情绪 |
| zoom_out | 拉远 | 展示环境 |
| pan_left | 左摇 | 场景转换 |
| pan_right | 右摇 | 展示全景 |
| tilt_up | 上摇 | 仰望视角 |
| tilt_down | 下摇 | 俯视视角 |

## 🎨 第三章：实战案例

### 案例：古风角色动态效果

#### 原图准备

:::step
**第一步**：生成高质量静态图

```bash
python jimeng_generate.py "古风美女提示词" 720 720
```

提示词要点：
- 完美精致五官
- 面部光照均匀
- 高质量动漫风格
:::

:::step
**第二步**：设计动作脚本

```
动作 1：眨眼 + 微笑（3秒）
动作 2：头发飘动 + 衣服飘动（5秒）
动作 3：转身 + 眺望（5秒）
```
:::

:::step
**第三步**：批量生成片段

使用不同的动作提示词生成多个片段，
然后在剪映中合成完整视频。
:::

## 💡 常见问题

### Q1：生成的视频卡顿怎么办？

:::warning 解决方案
1. 降低动作强度参数
2. 缩短视频时长
3. 使用更高质量的原图
4. 检查网络稳定性
:::

### Q2：角色变形怎么办？

:::warning 解决方案
1. 确保原图五官清晰
2. 使用更保守的动作参数
3. 选择稳定性更强的工具（如可灵AI）
:::
