---

## 📋 项目概述

**项目名称**：AI 动漫顶级艺术创作体系

**项目定位**：专业级 AI 动漫创作完整指南

**项目目标**：
- 提供一套能够照着一步一步制作出成品的完整创作步骤
- 每个部分都有专业的顶级内容
- 使用 Pretext 排版技术，确保内容清晰易读
- 各部分之间跳转专业化，形成完整的知识网络

**技术栈**：
- React 18 + Vite + TypeScript
- TailwindCSS 3
- Pretext 排版技术（https://github.com/chenglou/pretext）
- 后端：Node.js + Express + SQLite

## 🎨 Pretext 排版技术应用

### 布局原则

1. **清晰的层次结构**
   - 主标题 → 副标题 → 章节标题 → 小节标题
   - 使用不同字号、字重、颜色区分

2. **内容分组**
   - 相关内容放在一起
   - 使用卡片、边框、背景色分组

3. **视觉引导**
   - 使用箭头、编号引导阅读顺序
   - 重要内容高亮显示

4. **留白艺术**
   - 段落间留白
   - 模块间留白
   - 给眼睛休息空间

### Pretext 组件

```jsx
// 使用 pretext 进行精确布局
import { measureLayout } from '@chenglou/pretext'

// 内容块组件
<ContentBlock>
  <Title>章节标题</Title>
  <Subtitle>副标题</Subtitle>
  <Paragraph>正文内容</Paragraph>
  <TipBox>提示框</TipBox>
  <CodeBlock>代码示例</CodeBlock>
  <PracticeBox>实践练习</PracticeBox>
  <Navigation>跳转链接</Navigation>
</ContentBlock>
```
